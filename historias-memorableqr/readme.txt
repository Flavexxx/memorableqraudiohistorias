Plugin para grabar y compartir historias de audio en homenaje.

== Description ==
Historias MemorableQR permite a los familiares grabar y compartir historias en audio.

== Installation ==
1. Sube el plugin a tu directorio de plugins
2. Activa el plugin desde el menú de WordPress
3. Usa el shortcode [historias_memorableqr] en cualquier página o entrada
